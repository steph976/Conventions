Le code IMDG

Définition

Le code IMDG est ensemble de règles et de normes concernant le transport maritime de marchandises dangereuses en colis.

Au titre de la convention SOLAS et sauf exemptions, ses dispositions s'appliquent aux navires de plus de 500 UMS.

Ces dispositions établies par des commissions spécialisées de l'OMI ont une portée internationale.

Le code IMDG est transcrit en droit français dans la division 411.

Les dispositions du code IMDG concernent le transport de colis par les navires, mais les dispositions à terre (chargement, déchargement, transport par camion, train, etc...) sont pratiquement les mêmes, afin d'assurer un continuité logistique.

Sommaire

Le code IMDG se présente sous la forme d'un document rédigé en sept parties principales, régulièrement mis à jour par les commissions compétentes de l'OMI.

Pour chaque produit dangereux transporté par colis, le code IMDG fixe les dispositions concernant :

  • Classification des matières (par risque et par type)

  • Emballages, et contenants (fut, conteneur, bidon, citernes)

  • Étiquetage et repérage (affiches, symbole, couleurs)

  • Stockage à bord (Quantités limites, ségrégation, arrimage, larguage)

  • Transports (procédures, documents obligatoires)

Risques

Chaque produit dangereux est répertorié est classifié en fonction d'un risque initial (éventuellement complété par une classification par risques secondaires et induits) :

  • Classe 1 - Explosifs

  • Classe 2 - Gaz

  • Classe 3 - Liquides inflammables

  • Classe 4 - Solides Inflammables

  • Classe 5 - Comburants

  • Classe 6 - Toxiques et infectieux

  • Classe 7 - Radioactifs

  • Classe 8 - Corrosif

  • Classe 9 - Divers dangereux

Emballages

Le code IMDG fixe les contenants autorisés pour chaque type de produit. Il introduit des obligations de solidité, d'essais, de vérifications et de marquage pour chaque type de contenant :

  • Conteneur

  • Citerne

  • Bouteille

  • fut

  • Sac

  • carton

Etiquetage

Des étiquettes normalisées doivent être apposées sur les contenant autorisés de manière à bien montrer les risque 

Une étiquette supplémentaire faisant apparaitre deux numéros doit dans certains cas être ajoutée :

Numéro de risque

Numéro UN

Stockage à bord

Le stockage à bord doit être organisé selon des procédures définies par le code. Deux principes sont :

L'arrimage à bord

Ségrégation (séparation des produits)

Quantité limite

Procédures de Transport

Le code IMDG fixe ou propose également des procédures concernant la manutention, le chargement, le déchargement et les documents obligatoires.
