Emballages

Les emballages ont été catégorisé en 3 groupes :

  • -Groupe I : Danger élevé

  • -Groupe II : Danger intermédiaire

  • -Groupe III : Danger faible

Marquage

Le code IMDG fixe les contenants autorisés pour chaque type de produit. Il introduit des obligations de solidité, d'essais, de vérifications et de marquage pour chaque type de contenant :

  • Conteneur

  • Citerne

  • Bouteille

  • fut

  • Sac

  • carton

Un marquage prévoit un symbole indiquant que le contenant est aux normes du code IMDG

Conteneurs

Lorsqu'il sont utilisés pour transporter des produits dangereux en colis, il doivent cependant utiliser le même marquage.
